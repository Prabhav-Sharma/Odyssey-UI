Introduction

Odyssey UI is a one stop spot for some of the most used components in any UI, it's built with the idea to make the process of making a web-app as smooth and memorable as possible. The concept of varying user experiences is what inspired the name of the library. Optimizing for the needs of the different user journeys.

Getting Started

All you need to use this component library in your project is to copy & paste the below line of code into your project stylesheet file. 

@import url("/CSS/Components/Grid/grid.css");
